# Bernoulli

## Probability mass function

$$
  p^m(x \mid p) = p^x (1 - p)^{1 - x}
$$

$$
  \log{p^m(x \mid p)} = x \log{p} + (1 - x) \log{(1 - p)}
$$

$$
  x = 0,1
$$

## Diff x

離散確率分布は value による微分不可

## Diff p

$$
  \frac{\partial \log{p^m(x \mid p)}}{\partial p} = \frac{x}{p} - \frac{1 - x}{1 - p}
$$
